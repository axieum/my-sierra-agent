# my-sierra-agent

An example [Sierra AI][sierra] voice agent.

```sh
pnpm install
pnpm sierra add-workspace suncorp.sierra.ai
```

## Development

### `pnpm upload`

Uploads the agent to a [Sierra AI][sierra] workspace.

### `pnpm build`

Builds the [Sierra AI][sierra] voice agent.

### `pnpm watch`

Uploads the agent to a [Sierra AI][sierra] workspace and watches for file changes.

### `pnpm check`

Checks the validity of the [Sierra AI][sierra] voice agent.

### `pnpm test`

Runs the tests and watches for file changes.

### `pnpm coverage`

Runs the tests _once_ with coverage.

### `pnpm fix`

Lints and corrects errors in files.

[sierra]: https://sierra.ai/
